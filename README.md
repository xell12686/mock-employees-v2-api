# Mock Employees JSON Server

## Setup

Install dependencies:

```
$ npm install
```

## Development

Start server:

```
$ npm run start
```

## Urls:

- Local: http://localhost:5002
- Online: https://mock-employees-v2-api.herokuapp.com


## Deployment:

- main branch is auto deployed to heruko https://test-products-v2-api.herokuapp.com
